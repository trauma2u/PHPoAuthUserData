PHPoAuthLib
===========
The Forked Project of [Oryzone/PHPoAuthUserData](https://github.com/Oryzone/PHPoAuthUserData) for Korean Providers

Installation
------------

```json
{
    "repositories": [
        {
            "type": "git",
            "url": "https://github.com/trauma2u/PHPoAuthUserData"
        }
    ],
    "require": {
        "oryzone/oauth-user-data": "~1.0@dev"
    }
}
```
